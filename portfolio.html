Página Web Papigs

Detalles del proyecto

Papigs es una página web de comida rápida con especialidad en brochetas de papa y cerdo. Utilicé el editor de gráficos Adobe Ilustrator para hacer el logo y algunas imágenes que componen dicha página; principalmente compuesta por un sticky navbar, un menú tipo acordeón responsivo, condicionales, funciones y vectores para cambiar las imágenes de la comida al ser cliqueados en cada opción del menú.

Carcterísticas:

  • HTML5
  • CSS3 Avanzado
  • JavaScript Avanzado
  • jQuery
  • Bootstrap 5
  • Diseño responsivo
  • Adobe Ilustrator

    Página Web Simon Game

    Detalles del proyecto

    Esta página está basada en el juego Simon; de forma aleatoria va iluminando los cuadrantes de colores, y a la vez que se ilumina cada cuadrante emite un sonido propio. Después de esperar, el usuario debe ir introduciendo la secuencia mostrada en el orden correcto, ayudándose de su memoria visual y sonora.

    Empleo esencialmente jQuery, uso de funciones, generación de números aleatorios o random, animaciones y sonidos.

    Carcterísticas:

  • HTML5
  • CSS3 Avanzado
  • JavaScript Avanzado
  • jQuery
  • No es un diseño responsivo

    • pauroro.github.io/SimonGame/

    Página Web Drum Kit

    Detalles del proyecto

    Drum kit fue realizada principalmente para establecer el uso de Javascript avanzado. Son una serie de botones que al ser cliqueables directamente se genera un sonido, al igual que al presionar la tecla de la letra correspondiente del teclado.

    Esto se realizó por medio del método Event Listener de tipo click y keydown; uso de funciones, métodos y constructores de objetos, declaraciones switch y el método de reproducción de audio.

    Carcterísticas:

  • HTML5
  • CSS3 Avanzado
  • JavaScript Avanzado
  • No es un diseño responsivo

    • pauroro.github.io/drumkit/

    Página Web Dice Game

    Detalles del proyecto

    El contenido principal de esta página fue generado mediante el uso de Javascript. A la medida que se refresca, esta muestra dos dados, los cuales generan valores aleatorios revelando un ganador o un empate.

    Se desarrolló mediante la generación de números aleatorios o random, uso del condicional if, selección de elementos con querySelector, cambios de imagen mediante el uso del método setAttribute y la propiedad InnerHTML para cambiar el contenido HTML del texto h1 que inicialmente dice Refresh Me.

    Carcterísticas:

  • HTML5
  • CSS3 Avanzado
  • JavaScript
  • No es un diseño responsivo

    • pauroro.github.io/dicegame/

    Página Web Tindog

    Detalles del proyecto

    En esta página web hago uso principalmente de Bootstrap 5, para manejo del Navbar, Containers, Buttons, Carousel y Cards. De igual manera, utilizo la propiedad z-index para indicar el posicionado de un elemento, en este caso, una imagen.
