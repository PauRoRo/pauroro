    Web Page Simon Game

    Project details

    This page is based on the Simon game; randomly illuminates the color quadrants, and at the same time that each quadrant lights up, it emits its own sound. After waiting, the user should go introducing the sequence shown in the correct order, using his visual and sound memory.

    I used essentially jQuery, functions, random numbers, animations and sounds.

    Characteristics:

  • HTML5
  • Advanced CSS3

  • jQuery
  • It isn't a responsive design
